# Афиши.ру — Структура сайта

## Страницы

| Файл                  | Описание                                      |
|-----------------------|-----------------------------------------------|
| `index.html`          | Главная страница                              |
| `cinema.html`         | Каталог кино (26 фильмов, фильтры, отзывы)   |
| `concerts.html`       | Каталог концертов (12 концертов, фильтры)     |
| `venue-cinema.html`   | Страница кинотеатра (Синема Парк / Киноград)  |
| `venue-concert.html`  | Страница концертной площадки (Лужники / ВТБ и др.) |

## Общие ресурсы

| Путь                    | Описание                          |
|-------------------------|-----------------------------------|
| `css/style.css`         | Все общие стили сайта             |
| `js/data.js`            | База данных: фильмы, концерты, площадки, кинотеатры |
| `js/components.js`      | Компоненты: шапка, подвал, переключатель городов, модалка билетов |

## Изображения

| Файл                       | Описание                        |
|----------------------------|---------------------------------|
| `images/logo.svg`          | Логотип Афиши.ру                |
| `images/favicon.svg`       | Иконка сайта                    |
| `images/hall-stadium.svg`  | Схема стадиона (концерты)       |
| `images/hall-concert.svg`  | Схема концертного зала          |
| `images/hall-cinema.svg`   | Схема кинозала                  |

## Технологии

- Чистый HTML5 + CSS3 (без фреймворков)
- Vanilla JavaScript (ES6+)
- Google Fonts: Oswald + PT Sans + PT Serif
- SVG для схем залов и логотипа

## Города

Поддерживаются: **Москва**, **Волгоград** (и другие через переключатель городов).

## Функциональность

- ✅ Переключение городов с сохранением в localStorage
- ✅ Каталог фильмов (26 штук) с фильтрами по жанру, формату, кинотеатру
- ✅ Каталог концертов (12 штук) с фильтрами по жанру, цене, площадке
- ✅ Система отзывов с рейтингом (звёздочки 1-10)
- ✅ Покупка билетов: выбор места → схема зала → оплата
- ✅ Страница кинотеатра: расписание, описание, залы, контакты
- ✅ Страница концертной площадки: расписание, схема зала, контакты
- ✅ Адаптивная вёрстка
