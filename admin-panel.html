Как работает определение города

1. Посетитель открывает сайт → браузер делает запрос к ipapi.co/json/?callback=...

2. Сервис возвращает { city: "Volgograd", country: "RU" }

3. Название города ищется в CITY_MAP → переводится в русское название

4. Город сохраняется в localStorage['afishi_city']

5. Пользователь может сменить город вручную через кнопку «Ваш город» в шапке

⚠️ Если пользователь уже выбрал город вручную — IP-определение не срабатывает
