Управление рекламными блоками на сайте
Я
Яндекс.Директ / РСЯ
📋 Инструкция: Яндекс.Директ → РСЯ → Мои блоки → Создать блок → Копировать код → Вставить выше.
Первое поле — код для <head>, второе — для тела страницы.
G
Google AdSense
📋 Инструкция: Google AdSense → Рекламные блоки → Создать блок → Auto ads или Display → Скопируйте оба фрагмента кода.
После сохранения вставьте скрипт из <head> в каждую HTML-страницу сайта перед </head>.
