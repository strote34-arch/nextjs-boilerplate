# Определение «сделано хорошо» — утренний дайджест роликов для афи.рф

Каждый критерий — да/нет:

1. Собран список предстоящих мероприятий с афи.рф на ближайшие 14 дней от дня запуска, с рабочими ссылками на карточки мероприятий (кино, концерты, спектакли, выставки). Если сайт недоступен — это явно сказано в начале отчёта и приложена запасная подборка по кинопремьерам.
2. Для каждого мероприятия проверено, есть ли уже ссылка на ролик youtube.com/@afishiru в его карточке.
3. Для каждого мероприятия без ролика подобраны 1–3 релевантных видео на YouTube: официальный трейлер/промо, соответствующий именно этому фильму/концерту/спектаклю; не фанатские нарезки и не случайные видео.
4. Все ссылки на YouTube в отчёте рабочие (проверены web_fetch) и ведут на заявленный контент; сомнительные совпадения помечены «⚠️ проверить вручную».
5. Для каждого мероприятия без ролика подготовлен готовый текст для вставки в карточку: одна строка «Смотрите видео: <URL>», которую можно скопировать как есть.
6. Отчёт — один самодостаточный HTML-файл `digest.html` в /mnt/session/outputs/: сводка сверху, затем таблица «мероприятие → дата → категория → ролики → текст для вставки», всё на русском, читается без пояснений.
