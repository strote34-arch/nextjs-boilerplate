# Эталонный кейс 1

Прошлых эталонов не было — сюда сохраняется первый проверенный владельцем удачный дайджест:

- `input.md` — дата запуска и условия (какие мероприятия были на афи.рф в тот день)
- `expected.html` — сам дайджест, признанный удачным

Это базовая точка регрессии: перед продвижением новой версии агента прогоняем её и сравниваем с этим эталоном.
