  1. Список мероприятий афи.рф на 14 дней вперёд, рабочие ссылки на карточки

  3. Для карточек без ролика — 1–3 релевантных официальных видео с YouTube
  4. Все ссылки рабочие; сомнительные помечены «⚠️ проверить вручную»
  5. Готовый текст «Смотрите видео: …» для каждой карточки
  6. Один читаемый HTML-файл digest.html со сводкой и таблицей
