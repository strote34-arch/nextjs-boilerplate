💰 Монетизация afishi.ru

Анализ билетных систем, рекламные форматы, подписка АфишиПлюс, интеграция ЮKassa и Ticketscloud. Все варианты — с цифрами.

🎟️ Анализ билетных систем

Вывод: Для afishi.ru оптимальна стратегия «двойная интеграция» — Ticketscloud для концертов/театров (партнёрский виджет + 10% с продажи) и Kinoplan для кинотеатров (B2B интеграция). Плюс собственная система через ЮKassa для прямых продаж.

Подробный анализ систем

🏆

Ticketscloud — РЕКОМЕНДОВАН №1

Концерты · Театры · Фестивали

Что получаем: Встраиваем виджет на afishi.ru, выбираем мероприятия в каталоге TicketsCloud, получаем 10% сервисного сбора с каждого проданного билета. Никаких вложений.

✅ ПЛЮСЫ
  • Виджет встраивается одной строкой кода
  • 10% сервисного сбора — наш доход
  • Тысячи мероприятий по всей РФ
  • Принадлежит МТС — надёжность
  • API + webhooks + аналитика
  • Работа без backend с нашей стороны
  • Партнёрская программа через Адмитад
❌ МИНУСЫ
  • Базовая комиссия 4% от организатора нам
  • Ограниченный контроль над UX
  • Зависимость от МТС-политики
  • Мало мероприятий в малых городах
Билет 1000₽ → покупатель платит 1100₽ (наш сервисный сбор 10%) → 100₽ нам → 1000₽ организатору → Ticketscloud берёт 4% от 1000₽ с организатора Наш чистый доход: 100₽ с билета = 10%

Как подключить: Зарегистрироваться на ticketscloud.com как «билетный партнёр», получить API-ключ, встроить виджет.

🎬

Kinoplan — РЕКОМЕНДОВАН для кино

Кинотеатры

Специализированная система для кинотеатров. Большинство независимых кинотеатров РФ работают на Kinoplan. Интеграция через API даёт расписание + продажу билетов напрямую.

✅ ПЛЮСЫ
  • Специально для кино — лучший UX
  • Реальное расписание тысяч кинотеатров
  • Схема зала встроена
  • Интеграция с ЕАИС (официальный учёт)
  • Партнёрская API документация открыта
❌ МИНУСЫ
  • Нужен договор с каждым кинотеатром
  • Комиссия обсуждается индивидуально
  • Требует backend для интеграции
  • Долгое подключение (B2B договора)

💳 ЮKassa — интеграция платежей

🎯 Стратегия: своя продажа через ЮKassa = максимальный доход

Когда площадка (театр, концертный зал) заключает договор с afishi.ru напрямую — мы берём 5–8% с каждого проданного билета. Это в 2 раза выгоднее чем быть посредником Ticketscloud.

Схема подключения ЮKassa

Шаг 1 — Регистрация (1–3 дня)

  1. Зайти на yookassa.ru, создать аккаунт
  2. Выбрать тип: ИП или ООО (самозанятым не подходит для агрегации)
  3. Загрузить документы (паспорт, ИНН, выписка из ЕГРЮЛ)
  4. Пройти модерацию (1–3 рабочих дня)
  5. Получить: shopId и secretKey

Шаг 2 — Тарифы ЮKassa

Способ оплатыКомиссия ЮKassaНаша наценкаИтого с билета
Карта (visa/mc)2.8%+4.2%7%
СБП0.4%+5.6%6%
SberPay2.8%+4.2%7%
ЮMoney3.5%+3.5%7%
Наличные (терминал)0%+5%5%

Шаг 3 — Код интеграции (frontend)

<!-- Подключить ЮKassa виджет -->
<script src="https://yookassa.ru/checkout-widget/v1/checkout-widget.js"></script>

<script>
// При нажатии "Купить билет"
function buyTicketWithYooKassa(eventId, seats, price) {
  var totalAmount = price * seats.length;
  
  // 1. Запрос к нашему backend: создать платёж
  fetch('/api/create-payment', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
      eventId: eventId,
      seats: seats,
      amount: totalAmount,
      currency: 'RUB',
      description: 'Билеты на мероприятие'
    })
  })
  .then(r => r.json())
  .then(data => {
    // 2. Открыть виджет ЮKassa с полученным confirmation_token
    var checkout = new window.YooMoneyCheckoutWidget({
      confirmation_token: data.confirmationToken,
      return_url: 'https://afishi.ru/ticket-success?id=' + data.paymentId,
      error_callback: function(error) {
        console.error('Ошибка оплаты:', error);
      }
    });
    checkout.render('payment-form');
  });
}
</script>

<!-- Форма для виджета -->

Шаг 4 — Backend (Node.js)

const { YooCheckout } = require('@a2seven/yoo-checkout');

const checkout = new YooCheckout({
  shopId: 'ВАШ_SHOP_ID',
  secretKey: 'ВАШ_SECRET_KEY'
});

// POST /api/create-payment
app.post('/api/create-payment', async (req, res) => {
  const { eventId, seats, amount } = req.body;
  
  // Наша комиссия 7% уже включена в цену билета
  const payment = await checkout.createPayment({
    amount: {
      value: amount.toFixed(2),
      currency: 'RUB'
    },
    payment_method_data: { type: 'bank_card' },
    confirmation: {
      type: 'embedded',
      locale: 'ru_RU'
    },
    capture: true,  // Автоматическое подтверждение
    description: `Билеты: ${seats.join(', ')} на событие ${eventId}`,
    metadata: { eventId, seats: seats.join(',') }
  });
  
  res.json({
    paymentId: payment.id,
    confirmationToken: payment.confirmation.confirmation_token
  });
});

// Webhook — подтверждение оплаты
app.post('/api/yookassa-webhook', async (req, res) => {
  const event = req.body;
  if (event.type === 'payment.succeeded') {
    const { eventId, seats } = event.object.metadata;
    // Сохранить билеты в БД, отправить на email
    await issueTickets(eventId, seats.split(','), event.object.id);
  }
  res.status(200).send('OK');
});
