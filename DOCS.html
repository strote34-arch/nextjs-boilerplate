📋 Документация проекта Афиши.ру

Портал афиши — концерты, театр, кино, выставки, детские мероприятия. Многогородский сайт с панелью управления, Пушкинской картой, системой отзывов, трейлерами и рассылками.

При переносе анонса из концертов в театр (смена cat → theater) — событие появится на странице Театры и исчезнет с Концертов при следующем рендере.
